# ثيم Moonli — Brand Theme

> أول ثيم علامة فعلي فوق [الأساس المحايد](../foundations/brand-foundation.md). يُعيد إسناد التوكنز القابلة للثيم فقط، ولا يلمس بنية النظام ولا المكوّنات.

## قرارات الثيم (موثّقة)

1. **لون الفعل = أسود (`#1C1C24`)، لا الليموني.** في تصميم Moonli الأزرارُ والأيقونات سوداء، والليموني تعبئة علامة. فجعلنا `--color-accent` أسودَ لأنه يعمل في كل أدوار accent: تعبئة زرّ + **نصّ رابط + حلقة تركيز** — وكلها تحتاج تباينًا عاليًا على الأبيض. (لو جعلنا accent ليمونيًا فاتحًا لانكسر تباين الروابط/التركيز ≈1.1:1.)
2. **توقيع العلامة الليموني = توكن جديد `--color-brand`** للأسطح والتظليل **فقط** (هيرو، كروت علامة، تظليل نصّ). **ممنوع** استخدامه كنصّ على أبيض. النصّ فوقه دائمًا `--color-on-brand` (داكن).
   > هذا يوسّع [brand-foundation](../foundations/brand-foundation.md): ثيم قد يُدخل **لون تعبئة علامة** منفصلًا عن accent. القاعدة: accent لون *فعل* عالي التباين؛ brand لون *سطح*.
3. **استخدام الليموني بشُحّ (60-30-10):** أسطح علامة معدودة (هيرو + كتلة العطاء + كرت إثبات + تظليل) — لا ملء كل قسم. الجسم يبقى محايدًا.
4. **زوايا أكبر** تعبّر عن نبرة العلامة: `--radius-lg: 24px`.
5. **الخط:** الموقع يستخدم Poppins؛ تعذّر تحميله (CSP في المعاينة) فاستُخدم بديل نظام هندسي. عند التوفّر يُحمَّل Poppins محليًا.

## التوكنز (تُحقن على الجذر)

```css
:root[data-theme="moonli"], .theme-moonli {
  /* الفعل: أسود — تعبئة/نصّ/تركيز عالي التباين */
  --accent-solid:  #1C1C24;
  --accent-hover:  #000000;
  --accent-active: #000000;
  --accent-tint:   #EFEFF2;   /* tint محايد فاتح (لا لون مشبع فاتح) */
  --on-accent:     #FFFFFF;

  /* توقيع العلامة الليموني — أسطح/تظليل فقط، لا نصّ على أبيض */
  --color-brand:        #C6F24E;
  --color-brand-strong: #AEE635;   /* تمرير/تأكيد */
  --color-on-brand:     #15161A;   /* النصّ فوق الليموني (تباين ≈ 14:1) */
  --color-brand-subtle: #F2FBDD;   /* خلفية تلميح خفيفة بالعلامة */

  /* نبرة الزوايا */
  --radius-md: 12px;
  --radius-lg: 24px;

  /* الخط (بديل نظام؛ Poppins عند التوفّر محليًا) */
  --font-family-base: ui-rounded, "Segoe UI", system-ui, sans-serif;
}
```

## شروط مُحقّقة (qa-checklist / brand-foundation)
- `on-accent`/`accent` = أبيض/أسود ≈ 17:1 ✓ · `on-brand`/`brand` = داكن/ليموني ≈ 14:1 ✓.
- حلقة التركيز `--shadow-focus` (الحلقة المزدوجة) خارجيتها accent **أسود** → ظاهرة على الأبيض وعلى الليموني ✓ (لهذا accent أسود لا ليموني).
- لون واحد للفعل (أسود)؛ الليموني سطحٌ لا ينافسه على الأفعال ✓.

## توكنات جديدة يحتاجها هذا الثيم
`--color-brand` · `--color-brand-strong` · `--color-on-brand` · `--color-brand-subtle`.
هذه **توكنات علامة اختيارية**: لا تُستهلك إلا في المكوّنات/الأقسام ذات الطابع التسويقي (Hero، Card fill=brand، تظليل). الأنظمة/الجداول لا تستخدمها.
