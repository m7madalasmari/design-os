# قائمة الفحص — QA Checklist

> تُمرَّر على كل شاشة/مكوّن **قبل الاعتماد**. أي بند راسب = الشاشة مرفوضة حتى يُعالَج. الاستشهاد بالأرقام عند الرفض.

---

## 1. التوكنز والقيم
- [ ] لا hex/rgb مكتوب يدويًا — كل لون من [tokens](../foundations/tokens.md) (ممنوع #11).
- [ ] لا px/مسافة خارج السلّم (ممنوع #12، #13).
- [ ] المكوّنات تستهلك التوكنز الدلالية لا الأولية ([brand-foundation](../foundations/brand-foundation.md)).

## 2. التسلسل والأفعال
- [ ] هدف واحد وعنوان واحد للشاشة.
- [ ] إجراء `primary` واحد فقط (ممنوع #14).
- [ ] التسلسل البصري = تسلسل الأهمية (ممنوع #3).
- [ ] صفوف الأزرار لها أولوية واضحة.

## 3. المكوّنات والمحتوى
- [ ] كل مكوّن من `/components` (ممنوع #2).
- [ ] لا كروت مكرّرة لقائمة متجانسة (ممنوع #1).
- [ ] لا محتوى/أعمدة/حقول مُضافة لم تُطلب (ممنوع #5).
- [ ] لا تعشيش حاويات مفرط (ممنوع #4).

## 4. الحالات
- [ ] فارغة / تحميل / خطأ / لا نتائج معرّفة حيثما تنطبق (ممنوع #17).
- [ ] التحميل بهياكل عظمية تحفظ التخطيط (لا قفز).
- [ ] حالات التفاعل: hover / focus-visible / active / disabled.

## 5. RTL والعربية
- [ ] لا `left`/`right`/`margin-left`/`text-align:right` تخطيطي (ممنوع #8).
- [ ] الأيقونات الاتجاهية فقط معكوسة.
- [ ] الأرقام/البريد/الأكواد صحيحة الترتيب (اختبر برقم هاتف + اسم لاتيني داخل جملة عربية).
- [ ] لا ميلان عربي مصطنع ولا وزن خفيف (ممنوع #9).
- [ ] نظام أرقام واحد (ممنوع #10).

## 6. الطباعة والنصّ
- [ ] لا حجم تحت 12px.
- [ ] ارتفاع السطر العربي ملائم (≈1.7 للجسم).
- [ ] النصّ يتبع [ux-writing](../foundations/ux-writing.md): موجز، مهني، بلا مبالغة/إيموجي، أزرار بأفعال.
- [ ] مصطلح واحد لكل مفهوم (المعجم).

## 7. الوصول (AA — غير قابل للتفاوض)
- [ ] تباين النص ≥ 4.5:1، والعناصر/النص الكبير ≥ 3:1.
- [ ] حلقة تركيز ظاهرة على كل عنصر تفاعلي (ممنوع #15).
- [ ] هدف لمس ≥ 44×44px.
- [ ] إنجاز المهمة الأساسية بلوحة المفاتيح وحدها.
- [ ] HTML دلالي (`button`/`table`/`label`…) وعلاقات ARIA صحيحة.
- [ ] الحركة تحترم `prefers-reduced-motion`.

## 8. الصدق والثيم
- [ ] لا بيانات وهمية تبدو حقيقية (المبدأ 8).
- [ ] لا إخفاء بيانات بصمت على الجوال (ممنوع #16).
- [ ] لو ثيم: `text-on-accent`/`accent` ≥ 4.5:1، ولون مميِّز واحد فقط ([brand-foundation](../foundations/brand-foundation.md)).

## 9. مكافحة AI Slop
- [ ] لا زخرفة بلا وظيفة (تدرّجات/ظلال ثقيلة/أيقونات حشو/رسوم) (ممنوع #6).
- [ ] لو أزلت عنصرًا ولم تتأثر المهمة → احذفه (المبدأ 1).

---

**نتيجة الفحص:** اكتب البنود الراسبة بأرقامها، أو «اجتاز كامل القائمة».
