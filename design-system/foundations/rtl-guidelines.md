# قواعد العربية أولًا — RTL Guidelines

> العربية أولًا قرار معماري لا إصلاح لاحق ([design-principles](design-principles.md) §5). هذا الملف **إلزامي** لأي مكوّن أو صفحة.
> القاعدة الأمّ: **لا توجد كلمة `left` أو `right` تتعلّق بالتخطيط في الكود.**

---

## 1. الإعداد الجذري

```html
<html lang="ar" dir="rtl">
```
- `dir="rtl"` يقلب تدفّق المحتوى وترتيب الأعمدة وافتراضات المحاذاة تلقائيًا.
- `lang="ar"` يفعّل تشكيل الحروف والكسر الصحيح للأسطر.
- المحتوى اللاتيني المضمّن يُعزَل، لا يُغيَّر `dir` للجذر.

---

## 2. الخصائص المنطقية حصرًا (Logical Properties)

استبدل كل خاصية فيزيائية بنظيرتها المنطقية. هكذا ينعكس كل شيء تلقائيًا بلا كود خاص بالاتجاه:

| ممنوع (فيزيائي) | المطلوب (منطقي) |
|---|---|
| `margin-left / right` | `margin-inline-start / end` |
| `padding-left / right` | `padding-inline-start / end` |
| `left / right` | `inset-inline-start / end` |
| `text-align: right` | `text-align: start` |
| `border-left / right` | `border-inline-start / end` |
| `border-radius` بأربع زوايا | `border-start-start-radius` … |
| `float: left/right` | `float: inline-start/end` |
| `width` ثابت قد ينكسر | `inline-size` |

المحاذاة الافتراضية للنص دائمًا `start`. لا تكتب `text-align: right` ظنًّا أنه «العربي الصحيح» — إن قُلب السياق للاتيني كسر.

---

## 3. ما الذي يُعكَس وما الذي لا يُعكَس

العكس ليس «اقلب كل شيء». بعض العناصر تحمل معنى ثابت لا يتبع الاتجاه.

### يُعكَس (يتبع اتجاه القراءة)
- التخطيط كله: الشريط الجانبي، ترتيب الأعمدة، تدفّق العناصر.
- **الأيقونات الاتجاهية:** الأسهم (رجوع/تالي)، علامات `>` `<`، شريط التقدّم، أيقونة الإرسال، التراجع/الإعادة، المسافة البادئة (indent).
- مؤشّرات الترتيب والتنقّل بين الصفحات (السابق/التالي).
- المنزلقات (sliders) واتجاه تقدّمها.

### لا يُعكَس (معنى ثابت)
- **الأرقام نفسها** — تُكتب يسارًا-لليمين دائمًا (123 لا 321).
- الشعارات، الصور، الفيديو، وأزرار تشغيل الوسائط (▶ تشير لاتجاه التشغيل لا القراءة).
- المحور الزمني في الرسوم البيانية (الوقت/التطوّر يتبع اصطلاح البيانات، ويُوثَّق لكل رسم).
- الأكواد، عناوين URL، البريد، أرقام الهواتف.
- علامة الصح ✓، أيقونات غير اتجاهية (بحث، إعدادات، حذف).
- الأسماء التجارية اللاتينية.

> عند الشكّ: «هل تتغيّر دلالة الأيقونة لو قُلبت؟» إن نعم فهي اتجاهية تُعكَس، إن لا فتبقى.

### تقنية عكس الأيقونات الاتجاهية
```css
[dir="rtl"] .icon--directional { transform: scaleX(-1); }
```
طبّقها على الاتجاهية فقط — لا تعمّمها على كل الأيقونات.

---

## 4. ثنائية الاتجاه — Bidi (الأخطر)

عند خلط محتوى لاتيني أو أرقام داخل نص عربي، قد يُعيد المتصفح ترتيب الأجزاء بشكل خاطئ (مثلًا يبعثر رقم هاتف أو يفصل علامة عن رقمها).

**الحلّ — اعزل المحتوى المختلف اتجاهه:**
```html
<!-- نص عربي يحوي اسمًا لاتينيًا ورقمًا -->
<p>المستخدم <bdi>John Doe</bdi> سجّل دخوله في <bdi>10:45</bdi></p>
```
أو بالـ CSS لعناصر معروفة (حقل بريد، حقل URL، خلية كود):
```css
.ltr-content { direction: ltr; unicode-bidi: isolate; text-align: start; }
```

**قواعد:**
- البريد، URL، الأكواد، أرقام الهواتف، أسماء المستخدمين اللاتينية → اعزلها بـ`<bdi>` أو `unicode-bidi: isolate`.
- حقول الإدخال لهذه الأنواع: `dir="ltr"` على الحقل نفسه، مع بقاء التسمية والتخطيط RTL، والمؤشّر يبدأ من اليسار داخل الحقل.
- الرموز الملتصقة بالأرقام (٪، عملة): استخدم العزل ليبقى الرمز ملاصقًا لرقمه. صيغة العرض تُحدَّد في [ux-writing](ux-writing.md).
- الأقواس والترقيم: يتكفّل بها خوارزم bidi تلقائيًا داخل النص المعزول صحيحًا — لا تعكسها يدويًا.

---

## 5. الإدخال والنماذج

- اتجاه الكتابة للحقول العربية `rtl`، والمؤشّر يبدأ من اليمين.
- حقول البريد/URL/الأرقام `dir="ltr"` لكن تبقى التسمية ورسالة الخطأ ومحاذاة العنصر ضمن تدفّق RTL.
- علامة الحقل المطلوب وأيقونة الحالة توضع في الجهة المنطقية الصحيحة (`inline-start`/`end`)، انظر [form](../components/form.md).

---

## 6. الحركة والاتجاه

حركات الانزلاق تتبع الاتجاه المنطقي: الدخول من `inline-start` في RTL = من اليمين. عرّف الحركة بالمحور المنطقي لا بـ`translateX` ثابت الإشارة، أو اقلب الإشارة تحت `[dir="rtl"]`.

---

## 7. فحص سريع (يدخل [qa-checklist](../rules/qa-checklist.md))
- [ ] لا `left`/`right`/`margin-left` … في أي CSS تخطيطي.
- [ ] الأيقونات الاتجاهية فقط معكوسة؛ غير الاتجاهية سليمة.
- [ ] الأرقام والبريد والأكواد تظهر صحيحة الترتيب (اختبر برقم هاتف واسم لاتيني داخل جملة عربية).
- [ ] ترتيب التركيز بلوحة المفاتيح يتبع القراءة (يمين→يسار، أعلى→أسفل).
- [ ] الشريط الجانبي على اليمين، والمحتوى يساره.
