# صفحة هبوط «نبض» — System Fidelity test fixture

> **هذا fixture اختباري، لا صفحة منتج.** يثبت أن **System Fidelity Mode** ينقل صورة مرجعية (صفحة هبوط وكالة تسويق) إلى صفحة عربية RTL متوافقة مع النظام، دون نسخ بكسلي ودون كسر مصدر الحقيقة.

- **Type:** System Fidelity test fixture لـ **design-os v1.3.0**.
- **Not a product page** — بُني لتمرين البروتوكول، لا للنشر. لا يُنسخ لمشروع مستهلك (الـfixtures تبقى في `playground/`).
- **Purpose:** التحقّق من أن النظام + الثيم (moonli) يغطّيان صفحة هبوط تسويقية حقيقية (nav · hero + رسم · شريط مصادر · 4 كروت مزايا متناوبة · كتلة دعوة · 3 كروت حالات داكنة · تذييل) من النظام والثيم وحدهما.
- **Reference:** صورة مرفقة — صفحة هبوط وكالة تسويق رقمي (أبيض/أسود/أخضر ليموني، كروت متناوبة، تسميات أقسام مظلَّلة، رسوم خطية).
- **Resolved inputs (من المستخدم):** المجال = **منصة تحليلات SaaS**؛ المنقول من المرجع = **البنية + لوحة الأسود/الأخضر + الرسوم**.
- **Theme:** [`theme-moonli`](../../design-system/themes/theme-moonli.md) — أسود فعل + ليموني سطح علامة (هو بالضبط لوحة المرجع، وتباينه مُثبَت مسبقًا، فأُعيد استخدامه بدل توكن جديد).
- **Preview runtime:** **Shim mirror** — shim موثّق يطابق [`tokens.css`](../../design-system/tokens.css) + الكلاسات الدلالية 1:1 (لا توجد بنية Tailwind هنا). **ليس** التنفيذ الإنتاجي.

## القرارات الصادقة (مهمّة)
- **لا شعارات عملاء وهمية:** «جدار الشعارات» في المرجع (Amazon/Netflix…) → شريط **مصادر/تكاملات** صادق (SQL · API · CSV · Webhooks · جداول) = وصف قدرة لا ادّعاء رعاية.
- **لا أرقام تُعرض كحقيقية:** كروت «حالات الاستخدام» تحمل شارة ظاهرة **«الأرقام عيّنة توضيحية»**، وكلّ رقم موسوم «(عيّنة توضيحية)» — يحفظ طاقة «كرت الإحصاء» في المرجع دون تضليل.
- **اللون عبر ثيم لا hard-code:** الأسود/الليموني من moonli؛ الليموني سطح علامة فقط (تظليل/شارات/أسطح معدودة) — لا نصّ ليموني على أبيض. شُحّ 60-30-10.
